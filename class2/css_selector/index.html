What are Element Selectors?

Element selectors target HTML elements by their tag name.

They apply styles to all instances of that element on the page.

Benefits:

  • Simple and easy to use
  • Great for global styling
  • Low specificity makes them easy to override

Common Uses:

  1. Typography settings
  2. Default link colors
  3. Form element styling
